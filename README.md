# Weather app

V této aplikaci si můžeme zobrazit aktuální počasí, předpověď počasí, změnit jednotky, přidávat a
odstraňovat lokace.

## Vyuzite technologie:

- Koin - pro dependency injection
- Ktor - pro dotazovani k API (open-meteo)

## Co umoznuje?

- Current weather
- Weather forecast
- Search location
- Add location
- Remove location
- Edit weather units (temperature, speed, ...)
